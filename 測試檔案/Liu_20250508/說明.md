# 以下版本可以分別將兩張圖片的車牌辨識出來

## 版本一 success_91_97

### 步驟
1. 灰階轉換
2. 圖像切塊 → 局部平均
3. 二值化（區塊平均 - 誤差值）
4. 中值濾波 → 去小雜訊
5. 邊緣偵測：使用 Sobel 濾波
6. bwareaopen → 移除小連通區塊
7. imclose 補強橫向邊緣連線
8. regionprops → 尋找候選車牌框
9. imcrop + OCR 判斷(使用中值濾波完的結果)
10. 找出車牌位置


## 版本二 success_89_97

### 步驟
1. 灰階轉換
2. 圖像切塊 → 局部平均
3. 二值化（區塊平均 - 誤差值）
4. 中值濾波 → 去小雜訊
5. 邊緣偵測：Morphological Gradient
6. bwareaopen → 移除小連通區塊
7. imclose 補強橫向邊緣連線
8. regionprops → 尋找候選車牌框
9. imcrop (擷取某區域) -> imadjust (對比增強) -> imresize (放大擷取區域倍率提高字型辨識度)
10. OCR 判斷(使用中值濾波完的結果)
11. 找出車牌位置
